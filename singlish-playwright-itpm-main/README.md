## Project Title
Automated Testing of Singlish to Sinhala Transliteration System

---

## Objective
This project automates functional and UI testing for the Singlish to Sinhala
transliteration system available at:

https://www.swifttranslator.com/

The objective is to evaluate:
- Accuracy of Singlish to Sinhala conversion
- Robustness under incorrect or unusual inputs
- Real-time UI behavior during text input

The application performs automatic real-time conversion without a submit button.

---

## Testing Scope
The following test types are covered according to Appendix 2 of the assignment:

- Positive Functional Test Cases: 24
- Negative Functional Test Cases: 9
- UI Test Case (Real-time output update): 1

Total Automated Test Cases: 34
